使用tmux进行后台任务运行防止训练中断
```bash
# 1. 更新软件包列表（确保能装到最新版）
apt-get update

# 2. 安装 tmux
apt-get install -y tmux

# 3. 创建会话
tmux new -s <会话任务名>

# 4. 在新会话运行训练代码
python train.py

# 5. 退出当前会话
# 使用按键组合，ctrl+b，松开按d

# 6. 重新进入会话
tmux attach -t <会话任务名>

# 7. 训练完成后删除会话
tmux kill-session -t <会话任务名>
```
